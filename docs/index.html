OpenStore

Una creadora de tiendas virtuales basada en microservicios

OpenStore centraliza el control de clientes, tiendas, productos y autenticación con una arquitectura escalable y replicable para desarrollo y despliegue.

Bienvenida

Proyecto OpenStore

OpenStore es una plataforma para crear y administrar tiendas virtuales con control de clientes, catálogos y autenticación centralizada. El backend se organiza en microservicios para que cada dominio evolucione por separado y el sistema pueda desplegarse de forma reproducible.

React Vite TailwindCSS Spring Boot FastAPI Express + Prisma
