OpenStore Frontend

React + Vite + TailwindCSS

La interfaz de usuario está construida con React, servida por Vite y apoyada en TailwindCSS para acelerar la composición visual. La navegación se resuelve con React Router y el estado de autenticación vive en un contexto central.

Stack visual

Cómo se construye la interfaz

El frontend no es una sola vista monolítica. Las pantallas están separadas por ruta y por componente, así que la app puede crecer sin volver inmanejable el árbol de UI.

React 19 Vite TailwindCSS 4 React Router TypeScript

Renderizado

Cómo se renderizan las páginas dentro de la app

Rutas públicas

HomePage vive dentro de PublicLayout. Esa capa controla el header público, el cambio de tema y la navegación base de la landing.

Rutas protegidas

ProtectedRoute valida si existe sesión. Si no hay usuario autenticado, redirige a /login antes de permitir entrar a owner, admin o perfil.

Panel interno

AppLayout agrupa las páginas que usan sidebar y topbar: owner, admin y profile.

Storefront público

La ruta dinámica /:shopSlug?id=... carga la tienda, su tema y su catálogo para renderizar el template público adecuado.

Login

Cómo maneja el inicio de sesión

Owner y admin

El formulario de login envía email y contraseña al backend. La respuesta se guarda en AuthContext y en localStorage, y luego se redirige según el rol: OWNER al panel del dueño y ADMIN al panel de administración.

Cliente

Cuando el flujo ocurre dentro de una tienda, el gateway soporta las rutas con shopId para registrar o autenticar usuarios de tienda. Eso permite cubrir escenarios de cliente sin romper el login central.

Persistencia de sesión: el token queda en localStorage y la app vuelve a hidratar el usuario al recargar la página.

Componentes

La UI está separada por componentes reutilizables

Layouts

PublicLayout, AppLayout y el layout del storefront separan cada contexto visual.

AuthContext

Centraliza usuario, login y logout para que cualquier pantalla pueda leer la sesión.

ThemeConfig

Mantiene el tema activo y permite alternarlo desde la interfaz pública.
